# 📝 Notes App

A simple Notes App built using HTML, CSS and JavaScript.

## Features

- ➕ Add Notes
- ✍️ Edit Notes
- 💾 Save Edited Notes
- 🗑 Delete Notes
- 🔍 Search Notes
- 💾 LocalStorage Support
- 📱 Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
